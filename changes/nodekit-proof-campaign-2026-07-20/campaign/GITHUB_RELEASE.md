# NodeKit v0.2.1: proof-carrying campaign release

NodeKit v0.2.1 packages the production-proof campaign for the first complete
recursive dogfood run: exact product evidence, editable presentation, captioned
walkthroughs, claim ledger, and publication controls.

## What is proven

- NodeKit v0.2.0 generated and proof-verified two app presets from empty
  directories in a deterministic 37.221-second local acceptance run.
- Founder Quest Graph is deployed as a synthetic, read-only production journey.
  All 15 hosted checks passed in four isolated desktop/mobile light/dark browser
  contexts; a fresh Windows worktree verified 25/25 artifact hashes.
- The vertical and technical walkthroughs are bound to the exact Founder Quest
  deployment and content-addressed media receipts.
- The eight-slide carousel remains editable in PowerPoint and is exported as a
  verified PDF for LinkedIn.

## Deliberate boundaries

- Founder Quest performs no durable writes and no remote Neo4j writes.
- Synthetic progress is not a bank, legal, regulatory, scientific, or other
  external approval.
- The Agentic RL lab is a deterministic behavior-cloning baseline, not gradient
  reinforcement learning.
- The Casca FDE lab is local-ready, not production-proven, and is unaffiliated
  with Casca.
- Anthropic's rare-disease research grant is an independent scenario; NodeKit is
  not affiliated with or endorsed by Anthropic and has not received the grant.
- The npm package remains unpublished; this is a GitHub source and evidence
  release.
