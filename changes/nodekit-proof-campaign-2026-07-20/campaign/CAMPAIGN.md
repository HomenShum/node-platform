# NodeKit launches NodeKit — campaign contract

This campaign is a recursive dogfood run, not a permissionless social bot.
NodeKit may research, draft, render, upload, publish, and verify the exact
authorized campaign. It may not reply, direct-message, follow, vote, buy
promotion, or broaden the publication surface.

## Public thesis

Most coding agents stop when code exists. NodeKit carries one bounded product
journey through research, generation, deterministic evaluation, live-provider
testing, browser QA, deployment, presentation, distribution, and proof.

## Reality anchors

- exact Git commits and config/application hashes;
- public GitHub repositories and release tags;
- strict provider receipts, including blocked attempts;
- fresh-browser screenshots and traces;
- exact deployment identity;
- exported and reopened artifacts;
- public post URLs and screenshots.

7. the NodeSlide export/reopen receipt;

After publication, record the public URL, timestamp, screenshot hash, and the
rendered text/asset comparison. A draft preview is not a publication receipt.
